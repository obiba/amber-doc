OBiBa Amber Documentation
=========================

Targeted at individual studies and study consortia, `OBiBa <http://obiba.org/>`_ software stack (Amber, Opal, Mica etc.) provides a software solution for epidemiological data collection, management, analysis and publication. `Amber <http://www.obiba.org/pages/products/amber/>`_ is the data collection tool.
