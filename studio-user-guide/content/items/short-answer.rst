Settings

   :widths: 10 90
   :header-rows: 1

   * - Property
     - Definition
   * - ``Placeholder``
     - Example text included in the text area when it is empty.
   * - ``Hint``
     - Help text below the input field.
   * - ``Default``
     - The default value. Make sure it is in the right format.
